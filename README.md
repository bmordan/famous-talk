# Famous Talk 8th Oct 2015

## Getting started

Clone the seed project
```sh
git clone https://github.com/Famous/engine-seed
```
Install all the things
```sh
npm install
```
Start your rebuild service
```
npm run dev
```
View you changes at
```
http://localhost:1618
```
